# GlobalVet México

Sitio web estático listo para subirse a GitHub Pages.

## Estructura

- `index.html` inicio del sitio
- `pages/` páginas internas
- `css/` estilos generales y responsivos
- `js/` interacciones básicas
- `assets/` imágenes, logo y recursos visuales

## Publicación rápida en GitHub

1. Crea un repositorio nuevo.
2. Sube el contenido de esta carpeta.
3. Activa GitHub Pages desde la rama principal.
4. Si usarás dominio personalizado, agrega tu archivo `CNAME`.
